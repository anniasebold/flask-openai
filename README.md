## Descrição

API Desenvolvida com uma integração com OpenAI para responder a perguntas feitas por usuários.

## Pré requisito

É necessário ter o Python 3.9 instalado em sua máquina.

## Instalação

```bash
$ pip install requirements.txt
```

## Rodando a aplicação

Na raíz do projeto:
```bash
$ python3 run.py
```

## Testes unitários

```bash
# unit tests
$ pytest

# unit tests cases
$ pytest -s -v 
```
